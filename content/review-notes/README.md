# Interpretation Review Notes

These dossiers are indexed alongside the markdown source so QMD can retrieve past review decisions, recurring weak patterns, and rewrite guidance.

Recommended loop:

1. `npm run interpretations:audit:write`
2. Review the relevant family dossier and `latest-audit.md`
3. Use QMD to search for stronger neighboring examples and repeated phrasing
4. Edit markdown in `content/interpretations/`
5. Re-run build, audit, and tests
